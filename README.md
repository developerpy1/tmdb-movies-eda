# TMDB Movies EDA

Exploratory Data Analysis (EDA) of the TMDB Movies Dataset using Python and Pandas.

## Analysis

- Data inspection
- Data cleaning
- Revenue analysis
- Budget analysis
- Profit analysis
- Return on Investment (ROI)
- Movie rankings
- Advanced filtering
- Word frequency analysis
- Franchise analysis
- Director analysis
- Actor analysis
